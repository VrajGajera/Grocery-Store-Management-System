# 🛒 Grocery Store Management System

A full-featured Grocery Store Management System developed using ASP.NET Core and .NET. This application helps manage products, categories, billing, and customers with a simple and user-friendly interface.

---

## 🔧 Features

* Product Management (Add, Update, Delete)
* Category Management
* Billing System
* Customer Handling
* Clean and User-Friendly UI

---

## 💻 Technologies Used

* ASP.NET Core (.NET)
* C#
* HTML, CSS
* Entity Framework Core (MVC)

## 🚀 How to Run the Project

### 🔹 Step 1: Clone Repository

```
git clone https://github.com/your-username/Grocery-Store-Management-System.git
```

### 🔹 Step 2: Open Project

* Open **Visual Studio**
* Click on **Open a Project or Solution**
* Select `Grosary_net.csproj`

### 🔹 Step 3: Restore Packages

```
dotnet restore
```

### 🔹 Step 4: Run Application

```
dotnet run
```

OR

* Click ▶ Run in Visual Studio

### 🔹 Step 5: Open in Browser

```
https://localhost:xxxx
```
